# rooms.ts

1. Clone repository

   ```
   git clone https://github.com/mrinck/rooms.ts.git
   ```

2. Switch to cloned directory

   ```
   cd rooms.ts
   ```

3. Install dependencies

   ```
   npm install
   ```

4. Run server

   ```
   npm start
   ```

5. Connect to localhost:8080 via websocket client (e.g. wscat)

   ```
   npm install -g wscat
   wscat -c localhost:8080
   ```
